记录下自己博客的搭建过程

2019/05/23

从来没弄过博客这类东西，平时比较懒，也没有太多写日记写文章的习惯，不过第一天弄好博客（其实差的还多也就是借了别人的框架。。。。。。博客外观美化和功能优化在慢慢搞吧）怎么也需要写点啥。

这个博客是用HITHUB和HEXO搭建的，其实搭建前我还是比较小白的，搭建的过程中顺便增加了对HITHUB和HEXO的了解，目前的理解就是HEXO是一个博客模版框架，然后本地弄好后发布到HITHUB上，这样别人就可以看到自己的博客了。至于具体怎么搭建怎么配置网上教程大把也就不在此多说了。主要说下自己遇到的问题和解决方法。首先是修改配置文件的时候“：”后面要有一个空格，如果连着写就会报错。 其次就是在执行hexo -d遇到error: src refspec HEAD does not match any error: failed to push some refs to 'git@github.com:lyyan0523/lyyan0523.github.io.git’这个错误提示，百度了好多帖子也没有解决。最后发现是需要回到本地Hexo仓库。然后用命令git config –global 来设置用户名和邮箱。然后就可以顺利hexo -d了。其他的问题倒是基本没遇到，搭建得还算顺利。发现自己也没写出来啥干货~~~~

既然有了博客了以后就没事来这边写点啥，这个博客会一直更新下去的，在这里给自己再加加油~~！！！好好学习天天向上= 。=
