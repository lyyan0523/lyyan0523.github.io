记录下自己博客的搭建过程

技术心得 杂七杂八 2019/05/23

刚刚搭建好自己的博客，是在win10下利用github+hexo搭建的，虽然网上有很多类似的教程，不过自己决定还是记录下自己的博客搭建过程以及遇到的问题和解决方法。

第一步：下载并安装Node.js和git

首先把下载地址给大家提供出来方便朋友们下载。
Node.js下载地址：https://nodejs.org/en/download/
Git下载地址：https://git-scm.com/
安装这两个软件很简单，下载后无脑下一步（next）就可以。

第二步：安装配置Hexo

安装好node.js和git后就可以开始安装配置hexo了
在任意磁盘新建一个文件夹用来存放Hexo博客内容。我是创建了一个名为blog的文件夹，创建好文件夹后点击鼠标右键选择git bash here。弹出命令输入框后写入以下命令
cnpm install -g hexo-cli
hexo init
第一步是下载hexo，第二步是初始化hexo，这两步完成后文件夹就会自动生成hexo的文件

第三步：将Hexo部署到Github

1.登录github官网登录自己的账户，没有的话申请一个帐号然后登陆

官网地址：https://github.com

2.登录后新建一个仓库

create a repository

注意仓库名和用户名要一致，如我的用户名为lyyan0523,则仓库名为lyyan0523.github.io.
创建的同时勾选下面的Initialize this repository with a README
